Aprendiendo HTML5

Esta página está dedicada a explicar qué es HTML5, su estructura, sus etiquetas y su importancia en el desarrollo web. HTML5 es la versión más moderna del lenguaje HTML (HyperText Markup Language) y sirve principalmente para crear y estructurar páginas web. Es la base de casi todos los sitios que ves en internet.

Comenzar

¿Qué es HTML5?

HTML5 es la quinta versión de HTML (HyperText Markup Language), el lenguaje de marcado que estructura el contenido de las páginas web. No es un lenguaje de programación, sino un sistema de etiquetas que le dice al navegador qué es cada elemento (título, párrafo, imagen, video, etc.) para que lo muestre correctamente. Sirve para define la estructura y el semántica de una página con etiquetas como "header", "section", "article", "footer",permite insertar y reproducir audio, video, gráficos y animaciones de forma nativa, sin plugins externos,Facilita formularios, aplicaciones web offline, geolocalización y otras funciones mediante APIs de JavaScript integradas, Hace que los sitios sean compatibles con todos los navegadores y dispositivos, incluyendo móviles. Son importantes por que mejora la experiencia de usuario: carga más rápido, es responsive y funciona sin depender de complementos inseguros como Flash, Mejora el SEO y la accesibilidad: las etiquetas semánticas ayudan a buscadores y lectores de pantalla a entender el contenido y unifica el desarrollo: una base estándar que combina estructura (HTML5), presentación (CSS) y funcionalidad (JavaScript), facilitando sitios modernos e interactivos.

Estructura básica de HTML5

<!DOCTYPE html>
<html>
<head>
<title> Mi página</title>
</head>
<body>
</body>
</html>

🔴-DOCTYPE html: indicar al navegador qué tipo de documento y qué versión de HTML se está utilizando, para que el navegador renderice la página correctamente siguiendo las reglas del estándar.

🔴-html. definir la estructura y el contenido de una página web. En otras palabras, HTML no es un lenguaje de programación, sino un lenguaje de marcado que utiliza etiquetas para indicar al navegador cómo organizar y mostrar textos, imágenes, enlaces, tablas, formularios, videos, etc.

🔴-head Contiene metadatos y configuraciones del documento que no se muestran directamente en la página. Definir metadatos del documento (información sobre la página), Establecer el título que aparece en la pestaña del navegador, Vincular hojas de estilo externas o internas, Incluir scripts de JavaScript, Configurar codificación de caracteres, Optimización SEO mediante etiquetas (descripción, palabras clave, autor, etc.), Compatibilidad y adaptabilidad.

🔴-body. Define el contenido visible de una página web, incluyendo texto, imágenes, enlaces, videos y elementos interactivos, siendo esencial para la estructura y presentación del sitio.

Etiquetas principales

h1 a h6

Se utilizan para definir títulos o encabezados en una página web. h1 es el título más importante y h6 el menos importante.

p

Sirve para definir párrafos de texto. El contenido dentro de "p" se muestra como un bloque separado.

img

Se usa para insertar imágenes en una página web. Requiere el atributo src para indicar la ruta de la imagen y alt para el texto alternativo.

a

Define un enlace o hipervínculo. Utiliza el atributo href para indicar la URL a la que se dirige.

button

Crea un botón interactivo que puede ejecutar acciones, como enviar un formulario o activar un script.

ul / ol / li

ul: Lista no ordenada (con viñetas). ol: Lista ordenada (numerada). li: Elemento de lista, usado dentro de ul o ol.

HTML5 semántico

header

Se utiliza para contener el encabezado de una página o sección. Generalmente incluye el logotipo, el título y, en ocasiones, la barra de navegación principal.

nav

Define una sección de navegación que contiene enlaces a otras partes del sitio web o a páginas externas.

main

Representa el contenido principal y único de la página, excluyendo elementos repetidos como menús, barras laterales o pies de página.

section

Agrupa contenido relacionado dentro de una página, normalmente con un encabezado propio. Se usa para dividir el contenido en bloques temáticos.

article

Contiene contenido independiente y autocontenido, como una noticia, una entrada de blog o un comentario, que podría distribuirse de forma independiente.

footer

Define el pie de página de una página o sección. Suele incluir información de contacto, derechos de autor o enlaces adicionales.

Formularios en HTML5

Los formularios (el elemento `form`) son la forma en que una página web recoge datos del usuario: buscas, inicias sesión, envías comentarios… todo eso viaja dentro de un "form" al servidor cuando se “envía”. 🔴Elementos clave:🔴 1️⃣-"form": contenedor del formulario. Lleva atributos como `action` (a dónde se envía) y `method` (GET/POST). Agrupa los controles y define cómo se mandan los datos. 2️⃣-"label": etiqueta de texto asociada a un control. Ayuda a usuarios y a lectores de pantalla; al hacer clic en el label se enfoca el input vinculado (con `for="id"` o envolviéndolo). 3️⃣-"input": campo de entrada simple. Su comportamiento cambia con `type`: `text`, `password`, `email`, `checkbox`, `radio`, `number`, `file`, etc. Es el control más usado. 4️⃣-"textarea": caja de texto multilínea para comentarios o descripciones largas. Se define filas/columnas y el usuario puede escribir párrafos. 5️⃣-"select": menú desplegable de opciones. Cada opción va en un "option"; útil para listas cerradas (país, categoría). 6️⃣-"button": botón accionable. Dentro del form, `type="submit"` envía, `type="reset"` limpia, `type="button"` solo ejecuta JS. También existe "input type="submit", pero "button" permite contenido HTML dentro. Eso captura los datos y los manda al servidor para procesarlos.

Multimedia en HTML5

Explicacion de HTML5 con imágenes, audio y video.

Imágenes

🔴La etiqueta "img" se utiliza para insertar imágenes en una página web. Sus atributos más comunes son:🔴 1️⃣-src: especifica la ruta de la imagen. 2️⃣-alt: texto alternativo que describe la imagen (importante para accesibilidad). 3️⃣-width y height: definen el tamaño de la imagen. 4️⃣-title: texto que aparece al pasar el cursor sobre la imagen.

Audio

🔴La etiqueta "audio" permite reproducir archivos de sonido. Puede incluir controles para que el usuario reproduzca, pause o ajuste el volumen.🔴 🔵Atributos comunes:🔵 1️⃣-src: ruta del archivo de audio. 2️⃣-controls: muestra los controles de reproducción. 3️⃣-autoplay: reproduce el audio automáticamente (no recomendado sin interacción del usuario).

Video

🔴La etiqueta "video" se usa para insertar videos en HTML5.🔴 🔵Atributos comunes:🔵 1️⃣-src: ruta del archivo de video. 2️⃣-controls: muestra los controles de reproducción. 3️⃣-autoplay: reproduce automáticamente. 4️⃣-loop: repite el video. 5️⃣-poster: imagen que se muestra antes de reproducir el video.

Accesibilidad

🔴El atributo alt es fundamental para la accesibilidad web. Proporciona una descripción textual de la imagen para:🔴 1️⃣-Usuarios con discapacidad visual que usan lectores de pantalla. 2️⃣-Situaciones en las que la imagen no se carga. 3️⃣-Mejorar el SEO de la página.

Conclusión

HTML5 es la base que hizo la web moderna realmente práctica: unificó etiquetas semánticas ( "header", "article"…), trajo audio/video nativo sin plugins, canvas para gráficos, APIs de almacenamiento local, geolocalización, arrastrar-soltar y mucho más. Eso significa páginas más rápidas, accesibles y compatibles con móviles, menos dependencias externas y mejor SEO. En pocas palabras, HTML5 convirtió al navegador en una plataforma de aplicaciones completa; hoy desarrollar sin aprovecharlo es trabajar con una mano atada.
